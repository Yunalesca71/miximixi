## Context

当前项目是一个基于 React + Three.js 技术栈的现代饭店选择器应用。项目采用组件化架构，结合 3D 星空背景效果和 GSAP 动画库，提供沉浸式的用户体验。这是一个全新的功能，不涉及现有系统的修改。项目数据存储在组件状态中，使用 React Hooks 进行状态管理。

## Goals / Non-Goals

**Goals:**
- 提供沉浸式的 3D 星空背景体验
- 实现流畅的卡片滚动和缩放动画效果
- 支持响应式卡片展示，适配不同设备
- 使用现代 React 架构和 Three.js 3D 渲染
- 提供优雅的视觉效果和交互体验

**Non-Goals:**
- 不提供饭店推荐算法
- 不集成外部地图或评论服务
- 不支持用户账户系统
- 不提供社交分享功能
- 不使用 LocalStorage 持久化（数据存储在组件状态中）

## Decisions

### 技术栈选择
- **React 19 + Vite**：选择现代 React 框架和 Vite 构建工具，提供快速开发体验和优秀的性能
- **Three.js + @react-three/fiber**：用于创建 3D 星空背景效果，提供沉浸式视觉体验
- **GSAP**：专业的动画库，用于实现流畅的卡片滚动和缩放动画
- **CSS Grid + Flexbox**：用于响应式布局，确保跨设备兼容性
- **React Hooks**：使用 useState, useEffect, useRef 等现代 React 特性进行状态管理

### 架构设计
- **React SPA 应用**：基于 Vite 的单页应用架构
- **组件化设计**：将功能拆分为 App、RestaurantSelector 等独立组件
- **3D 背景集成**：使用 @react-three/fiber 将 Three.js 3D 场景无缝集成到 React 应用中
- **状态驱动**：使用 React Hooks 管理应用状态和副作用
- **响应式设计**：基于屏幕尺寸动态计算卡片参数

### 动画实现
- **GSAP 动画库**：用于高性能的卡片滚动和缩放动画
- **React Three Fiber**：实现多层星空的 3D 动画效果
- **滚轮交互**：支持鼠标滚轮控制卡片水平滚动
- **无限循环**：通过数组复制和位置计算实现无限滚动效果
- **动态缩放**：根据卡片距离视口中心的距离动态调整缩放比例

### 卡片视觉设计
- **现代卡片设计**：采用背景图片和渐变文字效果
- **3D 视差效果**：卡片悬停时的 3D 变换和光效动画
- **动态缩放**：基于滚动位置的实时缩放效果
- **中文字体集成**：使用本地自定义字体 `KeinannMaruPOP` 增强视觉效果
- **玻璃态按钮**：采用渐变背景和阴影效果的现代按钮设计
- **3D 星空背景**：多层星星动画创造深度感

## Risks / Trade-offs

**[风险]** 浏览器兼容性问题 → **缓解措施**：使用广泛支持的 Web API，避免实验性功能
**[风险]** 本地存储容量限制 → **缓解措施**：实现数据压缩和清理机制
**[权衡]** 功能丰富性 vs 简洁性 → **决策**：优先保证核心功能的简洁体验
**[权衡]** 动画效果 vs 性能 → **决策**：在低端设备上提供关闭动画选项
